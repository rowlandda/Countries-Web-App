This web-app was built using Angular 8 for reactivity and Bootstrap for the (admittedly minimal) styling. It consumes a public API from restcountries.eu and two sort-of-public APIs from Google Maps -- specifically Maps JavaScript and Geocoding APIs.

The app gets its country data by parsing the response from a GET request sent to https://restcountries.eu/rest/v2/all. The large clickable map on the initial page uses the Geocoding API, and the smaller map on the individual country overview is created using the country data in tandem with the Maps JavaScript API.
